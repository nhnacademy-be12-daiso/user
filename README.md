# User API

## 회원 관련

### 회원 가입

- [x] 회원 가입할 수 있는 기능을 제공합니다.
- [x] 이름, 연락처, 이메일, 주소, 생일 등을 포함해야 합니다.

---

### 회원 주소

- [x] 주소는 **최대 10개까지** 등록할 수 있으며 **별칭**을 설정할 수 있습니다.
- 예시
    - 별칭: `NHN`
    - 주소: `경기도 성남시 분당구 대왕판교로 645번 길 16`

---

### 회원 등급

- **3개월 이내 순수 주문금액**을 기준으로 산정합니다.
- 회원 등급은 `MyPage`에서 확인할 수 있으며, **회원 등급별 혜택**이 제공되어야 합니다.

> 순수금액 = 주문금액 - (쿠폰 + 배송비 + 취소금액 + 포장비)

#### 등급별 포인트 부여

| 등급   | 기준 금액       | 포인트 적립률        |
|------|-------------|----------------|
| 일반   | 10만원 이상     | 순수금액의 **1%**   |
| 로얄   | 10만원 ~ 20만원 | 순수금액의 **2%**   |
| 골드   | 20만원 ~ 30만원 | 순수금액의 **2.5%** |
| 플래티넘 | 30만원 이상     | 순수금액의 **3%**   |

---

### 회원 탈퇴

- [x] 회원 탈퇴 시 한 번 등록한 **아이디는 재사용할 수 없습니다.**
- [x] `MyPage`에서 회원 탈퇴 기능이 제공되어야 합니다.
- [x] 회원 탈퇴 시 **자동 로그아웃 처리**되어야 합니다.

---

### 휴면 회원

- [x] 최근 로그인 날짜 기준 **3개월 이상 미접속** 시 휴면 상태가 됩니다.
- [x] 휴면 상태에서 로그인 시 **인증 절차**를 통해 해지할 수 있습니다.
- [x] 인증은 **Dooray Message Sender**로 대체합니다.

## 포인트 관련

### 포인트 적립

- [ ] 포인트는 전체 도서에 대해서 기본 적립률을 설정할 수 있습니다.
- [ ] 도서 결제시 기본 적립률 기준으로 적립됩니다.
- [x] 회원가입시 5,000 적립됩니다.
- [ ] 리뷰작성시 500 적립됩니다.
- [x] 회원가입, 리뷰작성, 전체 도서에 대한 기본 적립율을 관리자가 수정할 수 있습니다.

### 포인트 적립/사용 이력

- [x] 회원은 mypage에서 포인트 적립/사용 내역을 확인할 수 있습니다.

### 포인트 결제

- [x] 포인트는 현금처럼 사용하여 결제할 수 있습니다.
- [x] 포인트는 전체 주문금액 또는 일부 금액을 결제할 수 있습니다.
